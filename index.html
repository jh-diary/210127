        하이퍼링크 태그- a 태그
        절대 경로, 상대 경로, 내부 파일들을 고려!
        절대경로는 타 웹사이트의 주소
        상대 경로는 웹사이트 안에서의 다른 페이지
        내부 파일은 현재 페이지 안의 다른 위치(id태그를 사용한 이동에서 사용)
        

          
 target 속성

          
 "_blank" 새로운 윈도우에서 새로운 페이지를 연다

          
 "_self" 현재 윈도우에 새로운 페이지를 적재한다.*디폴트 값임

        

        id 속성
        id속성을 이용한 내부 파일에서의 위치이동은  북마크 와 같이 실행된다.

        또한 a태그를 이용하여 이메일 및 다운로드 링크를 걸어둘 수 도 있다.

        base 태그- base태그는 헤드섹션에 base태그를 사용하면 모든 링크에 대한 기본 디렉토리를 지정할 수 있다.
        ex) head태그에 < base href="http://www.company.com/"/ > 이런 식으로 있으면
            body태그에 < a href="info.html" > 라고 했을 때 링크에 대한 기본적인 base로 head태그의 링크가 추가되는 것이다!
            이는 링크의 중복을 막을 수 있고 좀 더 간결하게 표현할 수 있다! 슬래쉬 파악 할 것!

        테이블태그- table,tr,td,th태그
        테이블의 하나의 행을 table row라고 해서 < tr > < /tr >로 표시하고 하나의 데이터는 table data라고해서 < tb > < /tb >라고 한다.
        tr태그는 td태그의 부모태그이다.
        테이블에 따라서 각 열에 헤더가 있는 경우가 있다. 그 경우는 table header의 약자인 th를 사용해서 만들어주면 된다. th태그는 볼드체, 중앙정렬로 표시된다.
        테이블의 경계를 지정하는 방법은 table 태그의 border 속성으로 지정하는데 border 속성의 값이 "1"이면 선이 생성되고 ""라면 생성되지않는다. 그 외의 스타일은 css로 적용한다.
        행병합(row span)과 열병합(column span)으로 속성의 값을 "2", "3" 등 갯수로 지정하면 현재의 셀 위치에서 값만큼의 행을 병합하겠다는 것이다.
        행병합은 rowspan="" 열병합은 colspan="" 으로 지정한다.
        테이블의 제목을 삽입하려면 < caption > < /caption >으로 지정한다.
